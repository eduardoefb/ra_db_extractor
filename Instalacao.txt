1 - Baixar os arquivos files.tar.gz e install.sh, e salva-los no diretório /tmp/
2 - Entrar com o usuário root:
   No caso do ubuntu, como o root, por padrão, não tem senha, executar o comando "sudo su -" ;

3 - Entrar no diretório onde os arquivos foram copiados:
   cd /tmp/
   
4 - Executar o script install.sh:
   chmod +x install.sh && ./install.sh
   Obs.: O script irá baixar todos os pacotes necessários para o funcionamento do servidor:  apache2, php, mariadb, etc.
   Caso o mariadb ou mysql já esteja instalado e existam dados importantes salvos nele, a instalação deverá ser feita manualmente, e não via script. Nesse caso, me avise porque ainda 
   não deixei o procedimento pronto para fazer manualmente;
   
5 - Durante a execuão do script, será pedida a nova senha para o usuário root (usuário root do mariadb, nã o usuário root do sistema linux). Cadastre uma senha de forma que possa ser lembrada
depois;

6 - Durante a execução, também serão populados no banco de dados, os dados antigos dos elementos (arquivo backup.sql no diretório database).  Após o término os dados estarão no banco, mas ainda
não será gerado nenhum arquivo .xlsx com as informações. Os arquivos somente são gerados quando são feitos uploads de novos logs para o servidor. Utilize os exemplos disponibilizados no google drive
para um teste.
